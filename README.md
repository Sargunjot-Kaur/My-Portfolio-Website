# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript.
https://sargunjotkaur.netlify.app/
This website presents my academic background, technical skills, projects, certifications, and contact information in a clean and professional layout.

## Features

- Responsive portfolio design
- Fixed navigation bar
- Smooth scrolling between sections
- Active navigation link highlighting
- Mobile burger menu
- Scroll reveal animations
- Downloadable CV button
- GitHub, LinkedIn, email, and contact links
- Project cards for showcasing work

## Sections

### Home

The Home section introduces me as a BEng Software Engineering student.  
It includes a short introduction, profile image, GitHub button, and CV download button.

### About

The About section includes my biography, education timeline, technical skills, certifications, and languages.

### Projects

The Projects section showcases my main projects:

- PIC Education Database System
- Password Strength Checker

### Contact

The Contact section includes links to email, LinkedIn, GitHub, and location.
